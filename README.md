## Overview

Pantry Panic is a recipe management web application designed to help users discover meals they can make using ingredients they already have. By leveraging the Spoonacular API, the app allows users to input pantry items and instantly generate recipe suggestions based on those ingredients.

This project was developed as part of a collaborative web development effort using Git and GitHub. My primary contribution focused on backend integration, including working with APIs and handling data flow between the application and external services.

> Note: The live demo is currently non-functional due to the API key being invalid.

**Tech Stack:**
- HTML, CSS, JavaScript  
- RESTful APIs & JSON  
- Git & GitHub  

## Showcase

Pantry Panic offers a simple, user-friendly interface with a clean and modern design focused on usability and accessibility.

**Key Features:**
- Pantry management system to track available ingredients  
- Ingredient-based recipe search powered by an API  
- Save favorite recipes using local storage  
- View detailed recipe instructions and information  
- Import/export saved recipes  
- Fully responsive design with mobile support
